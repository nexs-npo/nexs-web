Core Product Active

nexsアプリ

情報が増え続ける時代に、AIが「適切に減らす」ことで中立なシグナルを抽出できるかを検証。

コンセプト

ニュースキュレーションやSNS情報の過多により、本当に重要な「社会課題のシグナル」が埋もれています。nexs アプリは、AI が提示された課題情報から「本質的に重要な課題」を識別し、ユーザーに提示する実験プラットフォームです。

主な機能

  • • Signals：社会課題の兆候をキュレーション
  • • Knowledge：課題に関連する理論・仮説・データをリンク
  • • Projects：実装中のプロジェクト進捗を共有
  • • オープンソース化：すべてのコードを GitHub で公開

Research Items

  • • AIによる情報削減が偏りを抑えられるかの検証（Signals）
  • • エンジニア不在でもAIと協働した開発運用は成立するか
  • • ユーザーコミュニティの自律的な課題発掘プロセス構築
  • • 課題とアクション（プロジェクト）の接続メカニズムの検証

現在の進捗

Signals・Knowledge・Projects の3つの基本機能が実装完了。ユーザーテスト・フィードバック収集フェーズに入っています。

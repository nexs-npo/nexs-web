nexs Design Components

デザイン学習用コンポーネント集。各ページのコードを編集して、リアルタイムで変化を確認できます。

ページ一覧

About ページ

Hero セクション、Process Cycle（4ステップ）、Values セクション、Join セクション

学習対象：段落のタイプ分け、ステップボックスの色・数字、テキスト内容の修正

Signals ページ

ソートボタン、Signal アイテム（2つ）、スター数・コメント数表示

学習対象：ボタンのアクティブ状態、Signal カード内の要素配置、カウント表示

Knowledge ページ

カテゴリフィルター、Knowledge アイテム（3つ）、日本語タグ表記

学習対象：フィルターボタンのクリック動作、タグの色分け、アイテムカードの表示・非表示

Projects ページ

稼働中セクション、準備中セクション、Project カード、Research Items

学習対象：セクション分け、カードの詳細要素、ステータスインジケーター

📖 学習のコツ

  • • DevTools（F12）を開いて、各要素の class を確認してみてください
  • • テキストを変更してみて、どう見えるか確認しましょう
  • • 色の class を変更する場合は、他のページも確認して統一感をチェック
  • • 修正したコードは、スタッフに共有し、実装時の参考にします

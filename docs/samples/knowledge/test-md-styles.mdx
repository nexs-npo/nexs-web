## テキスト装飾

通常のテキストです。**これは太字（Strong）です。***これは斜体（Emphasis）です。*~~これは取り消し線（Strikethrough）です。~~

文中に`インラインコード（Inline Code）`を埋め込むテストです。背景色とフォントが変わります。

## リスト

### 箇条書きリスト (UL)
CSSでドットスタイルとインデントが定義されています。

* リストアイテム1
* リストアイテム2
    * ネストされたリストアイテム2-1
    * ネストされたリストアイテム2-2
        * さらにネストされたリスト（3階層目）
* リストアイテム3

### 番号付きリスト (OL)
CSSで数字スタイルとインデントが定義されています。

1.  手順その1
2.  手順その2
    1.  サブ手順2-1
    2.  サブ手順2-2
3.  手順その3

### チェックリスト (Task List)
CSS未定義。Astro標準でどう表示されるか確認します。

- [x] 完了したタスク
- [ ] 未完了のタスク
- [ ] これも未完了

## 引用 (Blockquote)

> これは引用文です。
> 複数行にわたる引用のテストです。
>
> > ネストされた引用（孫引用）はどのように表示されるでしょうか？
